- 在项目根目录下面新建 `.vnev` 文件夹
- pipenv install
- flask自定义命令
  - pipenv run flask forge 创建所有虚拟数据
  - pipenv run flask initdb --drop 删除数据库中所有表，然后重建
  - pipenv run flask init  初始化系统管理员
- pipenv run flask run
